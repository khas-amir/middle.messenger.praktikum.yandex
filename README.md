# Учебный проект "ЧАТ" в школе Яндекс Практикума

# Стек использованный в проекте
- `Parcel` (сборщик)
- `Pug` (для шаблонизации HTML)
- `PostCSS` (для стилей)


## Установка
- Установка зависимостей
```bash
    npm install
```
- Запуск dev окружения
```bash
    npm run dev
```
- Cборка проекта
```bash
    npm run build
```
- Запуск сервера nodejs
```bash
    npm run start
```
